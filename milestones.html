Milestones

Within the framework of project management, a milestone is an event that uses to mark specific point along the project timeline. These points may signal anchors such as a project start and end date, a need for external review or input and budget checks, among others. In many instances, milestones do not impact project duration. Instead, they focus on major progress points that must be reached to achieve success.
