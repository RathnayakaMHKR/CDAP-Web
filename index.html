MEDSWIFT

One of the major problems currently in the healthcare industry is that there is no proper way of recognizing Doctor's handwritings and transferring clinical records of a patient from one hospital to another in a secure way. This specially matters in emergency situations such as accidents. Furthermore, there's a huge probability of losing clinical data and making human errors when referring to the prescriptions.

"Medswift" addresses all the above problems by providing a secure and reliable web application. It's implemented using emerging technologies such as Convolutional Neural Network, k-nearest neighbors algorithms, blockchain mechanism, and InterPlanetary File System. The system recognizes doctor's handwritings using above machine learning algorithms and transfers clinical data across hospitals using Ethereum blockchain.
